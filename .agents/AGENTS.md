# Project-Scoped Rules for Modul PkM

## Panduan Gaya Penulisan
Untuk penulisan konten modul, Anda harus merujuk pada pedoman di file `panduan_penulisan.md` yang ada di root direktori.
Prinsip utama:
1. Tetap padat, ringkas (maks 3-4 kalimat per paragraf), dan menghindari bahasa yang terlalu dramatis/berlebihan ("lebay").
2. Pastikan narasi tetap mengalir menggunakan transisi antar subbab yang halus.
3. Selalu pertahankan format khusus LaTeX (misal: \textbf, \texttt) secara konsisten.
